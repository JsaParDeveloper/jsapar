Java Schema Parser


The JSaPar package provides a parser for flat and CSV (Comma Separated Values) files.

Example of reading CSV file according to an xml-schema:
Reader schemaReader = new FileReader("resources/xml/samples/01_CsvSchema.xml");
XmlSchemaBuilder xmlBuilder=new XmlSchemaBuilder();
Schema schema = xmlBuilder.build(schemaReader);
Reader fileReader = new FileReader("resources/xml/samples/01_Names.csv");
Builder docBuilder=new Builder();
Document document = docBuilder.build(fileReader, schema);
