JSaParSchema is the xsd schema that describes how a xml based JSaParSchema should be written

The actual schema file can be found at the address http://jsapar.tigris.org/JSaParSchema/1.7/JSaParSchema.xsd.

Use that address in your xml header if you have an xml editor that supports auto-completion.
Example xml file:
<?xml version="1.7" encoding="utf-8"?>
<schema xmlns="http://jsapar.tigris.org/JSaParSchema/1.7" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
   xsi:schemaLocation="http://jsapar.tigris.org/JSaParSchema/1.7 http://jsapar.tigris.org/JSaParSchema/1.7/JSaParSchema.xsd">
    <csvschema lineseparator=" ">
         <locale language="en" country="GB"/>
         <line linetype="Person" occurs="*" cellseparator=";">
         ...
